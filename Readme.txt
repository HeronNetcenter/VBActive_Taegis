Este é apenas o repositório de códigos para envio e hospedagem no GitHub.
Esta pasta não deverá ser usada para a alteração e compilação dos programas aqui armazenados.
O local correto para alteração e compilação dos códigos é no servidor NTCDW02 - C:\DWNetcenter\_Desenv\...

O processo para alteração, compilação e armazenamento no GitHub é:
	1. Localizar o código sob a pasta do servidor NTCDW02 - C:\DWNetcenter\_Desenv\...
	2. Efetuar as alterações necessárias e os testes.
	3. Após a homologação copiar a pasta inteira do programa sobrescrevendo o programa existente no repositório do servidor NTCDW02 - D:\REPOS\_Desenv...
	4. Usando o VS Code enviar o código para o GitHub usando o script git quick.
